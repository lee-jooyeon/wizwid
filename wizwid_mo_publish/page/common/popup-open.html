팝업 가이드

1. 팝업 호출 방법 - 자동실행 : <button data-target="layer_test">팝업실행</button> <a herf="javascript:void(0);" data-target="layer_test">팝업실행</a> * data-target 이름은 팝업ID와 동일 - 함수로 open 실행 : popup.openPopup('레이어ID'); - 함수로 close 실행 : popup.closePopup('레이어ID'); 2. callback함수가 필요할 경우 : popup.callBackPop('버튼ID','레이어ID', callback함수); - 버튼ID => callback함수가 실행되어야 하는 버튼 ID - 레이어ID => callback함수가 실행되는 버튼이 포함되어있는 레이어ID - callback함수 => 실행될 함수로 3. 팝업 노출 뒤 다른 팝업이 실행될 경우 순서 1. popup.openPopup('레이어ID') 순서 2. popup.callBackPop('버튼ID','레이어ID', callback함수); 순서 2-2. 호출할 callback 함수에 popup.openPopup('레이어ID')를 사용 * 해당 popup.openPopup('레이어ID')의 '레이어ID'는 새로 노출할 팝업 이름
